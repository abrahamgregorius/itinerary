An interactive Valentine's Day website featuring a retro 8-bit game-like adventure through memories, challenges, and heartfelt messages.

## Features

- **Home Page**: Retro pixelated entry point with floating hearts and 8-bit styling
- **Memory Quest**: Swipe-based card system with pixel art aesthetics
- **Love Challenge**: Interactive quiz game with retro feedback animations
- **Gift Reveal**: Cinematic envelope opening with pixelated love letter and special surprises

## Tech Stack

- React 18
- Vite
- Tailwind CSS
- Framer Motion for animations
- React Router for navigation
- Custom 8-bit CSS styling with Press Start 2P font

## Design Philosophy

- **8-bit Retro Aesthetics**: Pixel fonts, CRT scanlines, retro color palette
- **Desktop-Optimized**: Works perfectly on large screens with mouse/keyboard
- **Game-like Experience**: Button presses, pixel borders, retro sound cues
- **Emotional Journey**: Linear progression with unlockable romantic content

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Start the development server:
   ```bash
   npm run dev
   ```

3. Open [http://localhost:5173](http://localhost:5173) in your browser

## Build for Production

```bash
npm run build
```

## 8-Bit Features

- **Pixel Fonts**: Press Start 2P font for authentic retro feel
- **CRT Effect**: Scanline overlay for that classic monitor look
- **Pixel Borders**: Retro button styling with inset shadows
- **Retro Colors**: Limited palette with hot pink (#ff6b9d) and dark backgrounds
- **Flicker Effects**: Animated pixel glow and flicker animations
- **Desktop Friendly**: Large tap targets, keyboard navigation support

## Customization

The website content can be easily customized by editing the components in `src/components/`:

- `HomePage.jsx`: Entry point and hero section
- `MemoryQuest.jsx`: Memory cards and messages
- `LoveChallenge.jsx`: Quiz questions and answers
- `GiftReveal.jsx`: Love letter and final reveal content
